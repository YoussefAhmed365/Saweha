# saweha

Saweha is a real-time financial tracking app that helps families and groups manage shared expenses seamlessly. It allows users to track collective costs, maintain independent private budgets, and settle payments instantly with simple admin controls—all wrapped in a clean, user-friendly interface that works perfectly even offline.
